# scripts/experimental/

> Экспериментальные модули. Не используются в основном контуре системы.

| Файл | Назначение | Статус | Решение |
|------|-----------|--------|---------|
| `run_agent.py` | Автономный запуск агентов через Claude API | Архивирован (FC-03) | Claude Code - основная среда исполнения; API ограничен 8192 токенами и лишен MCP-инструментов |
| `contract_validator.py` | JSON Schema валидация выходов агентов | Отложен (FC-01) | Агенты выдают Markdown, не JSON; валидатор будет полезен когда/если перейдем на структурированные выходы |

Эти модули могут быть активированы в будущем при изменении архитектуры.
