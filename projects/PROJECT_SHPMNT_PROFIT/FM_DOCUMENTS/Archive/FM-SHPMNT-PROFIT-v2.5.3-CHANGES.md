## Применённые изменения (6)

### HIGH-001: Унификация срока дефицита
| Параметр | Значение |
|----------|----------|
| **Где:** | SHPMNT-055 (Таблица 23) |
| **Было:** | "дефицит 30+ дней" |
| **Стало:** | "дефицит 60+ дней" |
| **Причина:** | Противоречие с п.3.16.4 (возвраты 60 дней). Унификация сроков. |

### HIGH-002: Терминология эскалации
| Параметр | Значение |
|----------|----------|
| **Где:** | Параграф 82 |

| **Причина:** | Пропущено при замене терминологии в v2.5.2 |

### MED-001: Диапазоны актуальности НПСС

|----------|----------|
| **Где:** | Таблица 16, строка 1 |

| **Стало:** | "До 30 дней" |
| **Причина:** | Перекрытие диапазонов (следующий был "31-60 дней") |
