# ApplyAllChanges - Быстрый старт

## Что это?

Полностью автоматическая программа для применения **ВСЕХ 32 изменений** из `FM-LS-PROFIT-v1.2.1-AGREED-SOLUTIONS.md` к документу Word.

## Запуск за 3 шага

### Шаг 1: Перейти в директорию

```bash
cd PROJECT_SHPMNT_PROFIT/tools/NET/ApplyAllChanges
```

### Шаг 2: Запустить программу

```bash
./run.sh
```

### Шаг 3: Проверить результат

Откройте файл в Word:
```
PROJECT_SHPMNT_PROFIT/FM_DOCUMENTS/FM-LS-PROFIT-v1.2.1-DRAFT.docx
```

Включите режим рецензирования:
**Рецензирование → Показать исправления → Все исправления**

## Что делает программа?

✅ Создаёт бэкап (FM-LS-PROFIT-v1.2.1-DRAFT.bak)
✅ Применяет 4 КРИТИЧЕСКИХ изменения
✅ Применяет 13 ВАЖНЫХ изменений
✅ Применяет 10 СРЕДНИХ изменений
✅ Применяет 4 НИЗКОПРИОРИТЕТНЫХ изменения
✅ Обновляет метаданные (версия, дата, колонтитулы)

## Что будет в результате?

Все изменения видны в режиме рецензирования Word:
- **Автор**: Шаховский А.С.
- **Дата**: 29.01.2026
- **Режим**: Вставка (Insert)

## Если что-то пошло не так?

Откат из бэкапа:
```bash
cd ../../FM_DOCUMENTS
cp FM-LS-PROFIT-v1.2.1-DRAFT.bak FM-LS-PROFIT-v1.2.1-DRAFT.docx
```
