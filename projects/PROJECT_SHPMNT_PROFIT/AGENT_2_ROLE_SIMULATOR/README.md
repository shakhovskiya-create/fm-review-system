# ROLE_SIMULATION — Результаты симуляции ролей

> Папка содержит результаты работы Агента 2 (ROLE_SIMULATOR)

---

## Назначение

Здесь хранятся:
- Отчёты о симуляции ролей пользователей
- Найденные проблемы UX
- Способы обхода системы
- Рекомендации по улучшению

---

## Структура файлов

```
ROLE_SIMULATION/
├── README.md                    ← Этот файл
├── UX-FINDINGS-v2.5.3.md       ← Отчёт по версии 2.5.3

```

---

## Формат именования

```
UX-FINDINGS-v{VERSION}.md
```

Где `{VERSION}` — версия ФМ, для которой проводилась симуляция.
