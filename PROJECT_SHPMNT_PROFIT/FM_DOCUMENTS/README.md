# FM_DOCUMENTS — Хранилище функциональных моделей

Папка содержит все версии ФМ проекта «Контроль рентабельности отгрузок по ЛС».

---

## Структура

```
FM_DOCUMENTS/
├── README.md                           ← Этот файл
├── FM-LS-PROFIT-v[X.Y.Z].docx          ← Текущие версии ФМ
├── Templates/                          ← Шаблоны документов
└── Archive/                            ← Архив старых версий
    ├── FM-SHPMNT-PROFIT-v*.docx        ← Устаревшие версии
    └── *-CHANGES.md                    ← Файлы изменений
```
