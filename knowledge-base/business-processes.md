## О9. Продажи (ключевой для нашей ФМ)

### О9.2 Проектные продажи
Продажи на объекты строительства через партнёров.

**Ключевые термины:**
- **ТЗ (Техническое задание)** — заявка от клиента на комплектацию объекта
- **ОТР (Отдел технических решений)** — подбирает оборудование по ТЗ
- **ППО (Подбор правильного оборудования)** — результат работы ОТР
- **Объект строительства** — строительный объект, куда поставляется продукция
- **Проект** — совокупность ЛС для одного объекта
- **Партнер** — компания, через которую идёт продажа
- **Подрядчик** — организация, выполняющая работы на объекте

**Документы:**
- Локальная смета (ЛС) — ключевой документ для расчёта стоимости проекта
- Спеццены — специальные цены для проектных продаж

### О9.3 Возвраты
13-шаговый процесс возврата товара:
- IMS (инициация) → 1С:УТ (обработка) → ELMA (согласование) → Kafka (интеграция)
- Регламент: LS-RET в ФМ

### О9.4 Маркетплейсы
B2B-продажи через EKF Market и внешние площадки.

## О2. Ценообразование

### Виды цен в 1С:УТ
- **Базовая цена** — устанавливается документом «Установка цен номенклатуры»
- **Спеццена** — индивидуальная цена для клиента/проекта
- **НПСС** — нормативная плановая себестоимость (проект SBS)
- **Внутренняя цена** — трансфертная цена между юрлицами (проект SBS)

### Скидки
- Стандартные скидки по прайсу
- Проектные скидки (для проектных продаж)
- Целевая маржинальность и максимальные скидки (проект SBS-59)

## Закупки

### Импорт (ВЭД)
- Упаковочный лист → ПТУ (Приобретение товаров и услуг)
- ГТД, инвойсы, транспортные документы (проект SBS-129)
- ТТК (транспортно-таможенный коэффициент) по странам (проект SBS-58)

### Внутренний приход
- ERP (производство) → УТ (торговля) по внутренним ценам (проект SBS-130)

## Закрытие месяца (SBS-57)

6-шаговый процесс:
1. Установка дат запрета по группам пользователей
2. Восстановление последовательности партионного учёта
3. Расчёт НПСС
4. Расчёт фактической рентабельности
5. Перенос незавершённых отгрузок в рабочий период
6. Установка общей даты запрета

## Связь процессов с нашей ФМ

| Процесс | Как связан с FM-LS-PROFIT |
|---------|---------------------------|
| О9.2 Проектные продажи | Прямо: ЛС, спеццены, согласование отгрузок |

| О9.3 Возвраты | Влияет на итоговую рентабельность ЛС |
