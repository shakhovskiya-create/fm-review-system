# База знаний EKF — для агентов fm-review-system

> Эта база содержит контекст о компании, проектах и бизнес-процессах EKF.
> Агенты используют её при необходимости уточнения бизнес-контекста.

## Как использовать

Агент читает нужный файл из `knowledge-base/` когда:
- Нужен контекст о смежных проектах/системах
- Нужно понять бизнес-процесс компании
- Нужно проверить совместимость с другими модулями
