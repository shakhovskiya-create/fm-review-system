# Структура проектов

Каждая ФМ = отдельный проект в `projects/PROJECT_[NAME]/`:

```
PROJECT_[NAME]/
  README.md, CONFLUENCE_PAGE_ID, PROJECT_CONTEXT.md, WORKPLAN.md
  CHANGES/FM-*-CHANGES.md
  AGENT_1_ARCHITECT/ ... AGENT_5_TECH_ARCHITECT/
  REPORTS/
```
