# Правила редактирования ФМ

## Формат документа
- Нельзя менять структуру/оформление/таблицы/нумерацию без явного указания
- Можно менять содержание (текст, правила, логику)
- Можно добавлять пункты в существующем формате
- Если нужно изменить формат - СПРОСИТЬ через AskUserQuestion

## Текст и форматирование
- Дефис (-) вместо длинного тире. Пример: "Заказ - это документ"
- Буква "е" вместо "ё". Пример: "Отчеты", "Расчеты"
- Точка с запятой (;) в перечислениях, точка (.) в конце последнего
- Только русский язык, без англицизмов
- Только цифровая нумерация (1., 1.1., 1.1.1.)
- Простой язык. Тест: "Менеджер продаж это поймет?"

## Версионирование
- Формат: X.Y.Z (X - мажорная, Y - минорная, Z - патч)
- ВСЕГДА проверять версию через Confluence API, не из локальных файлов
- Одна дата = одна версия (если правки в тот же день - дополнить описание)

## Запрет упоминания ИИ
НИКОГДА не писать: Agent, Claude, GPT, ИИ, AI, Bot, LLM.
- Автор: ВСЕГДА "Шаховский А.С."
- Документ выглядит как написанный человеком
