# /evolve - самоулучшение агентов

Полные инструкции: прочитать `agents/EVOLVE.md` и выполнить описанный алгоритм.

## Краткий алгоритм

1. Прочитать все патчи из `.patches/` (кроме README.md, EVOLUTION_LOG.md)
2. Группировать по агенту и категории
3. Для каждой группы: определить правило, которое нужно добавить/изменить
4. Показать пользователю предложения через AskUserQuestion
5. После одобрения: обновить `agents/AGENT_*.md`
6. Записать результат в `.patches/EVOLUTION_LOG.md`
7. Перенести обработанные патчи в архив
